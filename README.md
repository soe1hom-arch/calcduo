# CalcDuo

**Dual Calculator for Android** — Run two calculators side-by-side on one screen.

![Android](https://img.shields.io/badge/Android-8.0%2B-brightgreen?style=flat-square)
![License](https://img.shields.io/badge/License-Apache%202.0-blue?style=flat-square)

---

## ✨ Features

- **Dual Calculator** — Two calculators side-by-side simultaneously
- **Multiple Tabs** — Create and switch between calculator instances
- **Sliding Drawer** — Easy tab management
- **Material Design 3** — Dark theme, smooth animations, haptic feedback
- **Advanced Functions** — Square root, power, percentages, π, e, and more
- **Fully Offline** — All calculations done locally, no internet required

## 🛠 Build

```bash
git clone https://github.com/soe1hom-arch/calcduo.git
cd calcduo
./gradlew assembleDebug
# → app/build/outputs/apk/debug/app-debug.apk
```
