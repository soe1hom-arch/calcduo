# CalcDuo - Dual Calculator App

A modern, professional Android calculator app with split-screen multi-calculator support.

## Features

- **Dual Calculator**: Run two calculators side-by-side simultaneously
- **Multiple Tabs**: Create multiple calculator instances
- **Sliding Drawer**: Easy access to manage all calculator tabs
- **Modern Design**: Material Design 3 with dark theme
- **Advanced Functions**: Square root, power, percentages, constants (π, e), and more
- **Fully Offline**: No internet required, all calculations done locally
- **Professional UI**: Haptic feedback, smooth animations, modern color scheme

## Building

```bash
./gradlew assembleDebug
```

The APK will be at `app/build/outputs/apk/debug/app-debug.apk`.

## Tech Stack

- **Language**: Kotlin
- **UI**: Material Design 3, ViewBinding
- **Architecture**: MVVM with ViewModel
- **Build System**: Gradle with Kotlin DSL
